# Archived: Identity Verification

This document is intentionally archived. Identity verification is no longer part
of the active product direction.

The current product is a personal job-application tracker:

- Email/password accounts.
- User-owned application data.
- CV parsing without storing uploaded files.
- Live Platsbanken search through JobTech.
- Export and account deletion for portability and privacy.

Do not implement identity-verification flows unless the product direction is
explicitly changed again.
