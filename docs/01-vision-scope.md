# Vision & Scope

## Vision

Ansokt helps job seekers keep calm, complete control over their job search. It
replaces the private spreadsheet many people build for applications, deadlines,
contacts, interviews and follow-ups.

## Scope

The product is a personal application tracker with:

- Email-based accounts.
- A kanban board for application status.
- Manual application rows from any source.
- Live Platsbanken search through JobTech.
- CV parsing, editable structured CV data and explainable skill matching.
- Timeline notes per application.
- CSV export and account deletion.

## Out Of Scope

- Employer-side recruiting workflows.
- Authority or partner integrations.
- Identity verification.
- Stored CV uploads.
- Analytics or advertising tracking.

## Success Criteria

- A new user can understand the product from the first screen.
- A user can add or save the first application in under a minute.
- Follow-ups and deadlines are visible without manual spreadsheet scanning.
- The user can export or delete their data without contacting support.
